REST API для управления библиотекой на Django Rest Framework.  
Позволяет вести учёт книг, авторов, пользователей и записей о выдаче книг.

## Технологии
- Python 3.10, Django 4.2, DRF
- JWT-аутентификация (Simple JWT)
- PostgreSQL
- Docker, Docker Compose
- drf-spectacular (OpenAPI 3.0)
- django-filter

## Функциональность
- CRUD авторов и книг
- Поиск и фильтрация книг по названию, автору, жанру, году
- Регистрация и аутентификация пользователей (JWT)
- Выдача книг: создание записи, автоуменьшение количества доступных экземпляров
- Возврат книги: специальный эндпоинт для администратора, восстанавливает экземпляр
- Разграничение прав: администратор (is_staff) – полный доступ, обычный пользователь – только чтение каталога и свои выдачи
- Автогенерируемая документация Swagger (OpenAPI)
